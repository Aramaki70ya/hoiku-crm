# E2Eテスト（面接件数モーダル）

## 準備

1. **テストデータの投入**
   ```bash
   npm run test:seed
   ```
   吉田担当のテスト求職者「テスト 花子」と今月の面接1件が作成されます。

2. **Playwright ブラウザのインストール（初回のみ）**
   ```bash
   npx playwright install chromium
   ```

3. **開発サーバー**
   別ターミナルで `npm run dev` を実行しているか、テスト実行時に `reuseExistingServer: true` で既存の localhost:3000 を利用します。

## 実行

```bash
npm run test:e2e
```

## 証明する内容

- **テスト1**: ダッシュボードで吉田の「面接」の数字をクリック → モーダルが開き、一覧に「テスト 花子」が表示される
- **テスト2**: モーダル内の「無効化」ボタンをクリック → 確認ダイアログを accept → 処理が完了する（一覧が空になるかモーダルが閉じる）

## UIで手動確認する場合

1. `npm run dev` で起動
2. `npm run test:seed` でテストデータ作成
3. ブラウザで http://localhost:3000/dashboard-summary を開く
4. 集計期間を「今月」にし、営業進捗テーブルで**吉田**の**面接**の数字をクリック
5. モーダルに「テスト 花子」が表示されることを確認
6. 「無効化」をクリックして確認ダイアログでOK → 件数から除外されることを確認
